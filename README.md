# Frontend Portfolio

A modern and responsive personal portfolio website built to showcase my skills, projects, education, certifications, and achievements.

## Features
- Responsive design
- Modern and clean UI
- Smooth navigation
- Projects showcase
- Skills section
- About Me
- Contact section

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Structure
```
frontend-portfolio/
├── index.html
├── css/
├── js/
├── images/
└── README.md
```

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/menika-kumari/Portfolio-FRONTEND.git
```

2. Open the project folder.

3. Run `index.html` in your browser.
